Welcome to AutodiffLabs Docs

This is a modern documentation framework built with React that replicates the simplicity of Docsify, now powered by raw HTML files.

How it works

  1. Content is written in HTML files.
  2. The sidebar is configured via _sidebar.md for easy structure management.
  3. Client-side routing fetches the HTML and injects it securely.

Key Features

  • Nested Sidebars: Supports infinite nesting levels with collapsible menus.
  • Client-Side Routing: Fast navigation without page reloads.
  • Full Text Search: Indexing runs entirely in the browser.
